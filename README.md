# paygo-uganda-customer-churn-dataset
Anonymized customer dataset for Explainable AI Customer Churn Prediction in PAYGo Solar (Uganda).   Used for capstone project: "Explainable AI for Predicting Customer Churn in Uganda’s PAYGo Solar Sector"

This project focuses on building a customer churn prediction and behavioral analytics system using loan repayment, payment behavior, and call center interaction data.

The goal is to:

Identify customers at risk of churn
Understand payment behavior patterns
Measure engagement using call activity
Build predictive and explainable insights for business decision-making
